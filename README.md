MLOps REST API System

1. Project Overview

This project is a RESTful API system developed for managing Machine Learning Operations (MLOps). It allows users to manage Workspaces, Machine Learning Models, and Evaluation Metrics.

The system is built using Java with JAX-RS (Jersey) and runs on a Grizzly HTTP server. It uses in-memory data structures (HashMap and ArrayList) for data storage.

The API follows REST architectural principles and supports full CRUD operations.



 2. Technology Stack

- Java (JDK 17)
- JAX-RS (Jersey Framework)
- Grizzly HTTP Server
- Maven (Build Tool)
- Postman (API Testing)
- In-memory Data Storage (HashMap / ArrayList)
